# simple_cli_rpg

日本語 | [English](./README.md)

ゲームループや状態管理にフックすることで、「ゴッドモード（無敵）」や「スピードハック」といった一般的なチート機能を実装するデモンストレーションです。

## The Target

シンプルなコマンドライン(CLI)ベースのロールプレイングゲームです。

**意図的なストレス** と固定されたパラメータで設計されています。

- **Stack:** Python標準ライブラリ (random, time)
- **Behavior:**
  - **遅いゲームプレイ:** アクション（探索、戦闘）ごとに強制的に1秒間の待機時間 (`time.sleep`) が発生し、テンポが悪く作られています。
  - **標準的な難易度:** 戦闘フェーズでは敵からダメージを受け、HPが減ります。
  - **固定されたID:** プレイヤー名はハードコードされており（例: "Hero"）、引数等で変更することはできません。

## The Mod

Modはゲームのルールと内部状態を改変し、ストレス要素を排除して難易度を無効化します。

- **プレイヤー名カスタム (custom_player_name.py):**
  - 実行時にプレイヤーの `name` 属性を上書きします。
  - 既存オブジェクトへの動的な状態注入を実証しています。
- **スピードハック / No Sleep (no_sleep_time.py):**
  - 遅延処理関数にフックします。
  - 人工的な1秒間のクールダウンを完全にバイパスし、ゲームプレイを即時応答・爆速化させます。
- **ゴッドモード / ダメージ無効化 (zero_damage.py):**
  - 戦闘ダメージ計算ロジックにフックします。
  - 敵からの被ダメージを強制的に `0` に固定し、プレイヤーを無敵状態にします。

## How to Run

### Prerequisite

`uv` がインストールされ、依存関係が同期されていることを確認してください。

### 1. Run Modded (Cheated)

ローダーがゲームにチートModを注入します。

```bash
cd examples/simple_cli_rpg
# Run the target via the Universal Modloader
uv run loader.py
```

**Try this:**  
ゲームをプレイしてください。アクション間の遅延がなくなり、プレイヤー名が変更され、戦闘でダメージを受けないことを確認してください。

### 2. Run Vanilla (Original)

本来の遅く、ダメージを受ける挙動を確認します。

```bash
cd examples/simple_cli_rpg
uv run main.py
```

**Try this:**  
入力のたびに発生する1秒間の遅延と、攻撃を受けるとHPが減ることを確認してください。
